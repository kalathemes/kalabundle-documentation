    Home Page Sections

    Overview

    From Sections tab of Customize (Theme Editor) You will see all sections of the viewing page.

    We will guide all sections in Home page of KalaBundle #001 (Fashion) v5.1, other Home style have different sections, but they also easy to config.

    Except 3 sections Header, Mobile Menu, Footer will appear as general sections in all pages, other sections in the red rectangle of above screenshot are using for Home Page only.

    How to Edit section

    Click to section that you want edit, the section will explain to show all sections to control it.

    How to Add new section

    At the bottom of Sections tab, click to add new section.

    How to Change position (re-order) section

    Click to the icon and hold it to move the section to area that you want.

    How to Delete section

    Click to section that you want delete, the section will explain, scroll down and click button to remove this section.

    How to Hide section (temporary don't use)

    Click to the icon, this section will disable. Click again to show/enable this section.

    *Screenshots from other theme (Marco) to demo how to use the Section function of Shopify.



    Sections listing

    The KalaBundle #001 v5.1 layout have 4 areas: Top Bar, Left sidebar, Right content and Bottom area. We will guide Top bar and all sections in Right content first, then 1 Left sidebar section and 3 Bottom sections.

    Welcome Message

    • Area: Top bar

    • Section Title as add new: Welcome Message in Store information group

    • Section Title as existed in demo: Welcome Message. In default (demo layout) this section only show in Mobile.

    Click to edit.

    These option fields for this section: Display on, Title (Heading), Sub Title and Content is HTML field.



    Slideshow

    • Area: Right content

    • Section Title as add new: Slideshow in Image group

    • Section Title as existed in demo: Slideshow

    Click to edit.

    These option fields for this section: Display on, Slide effect (include 4 effects: Sliding, Fading, Cube, Flipping ), Auto-rotate slides, Slide width in px (Desktop), Slide height in px (Desktop), Enable Navigaiton, Navigation Position and Enable Pagination.

    The Important area is the Content area (bottom). It will listing all slide of Slideshow. Click to Slide to edit. If you want add more slide, click to

    You will got options:

    • Slide Background: Upload image to show. Also can set Slide link, Captions Entrances Effect and Captions Exits Effect

    • Title: It's heading of the caption.

    • Description: it's description of the caption.

    • Action: it't action button of the caption.

    • Other captions (maximun is 6) you can set below.

    With each caption element, you need set the style and position to make it show the like your idea. These options area Style, horizontal position(x), align with horizontal position, vertical position(y), layer order(z)



    Top Collection

    • Area: Right content

    • Section Title as add new: Top Collection in Collection group

    • Section Title as existed in demo: Top Collection

    Click to edit.

    These option fields for this section: Display on, Title (Heading), Sub-title, Image width, Image height, Items cut off point (It will only show x_number collections only, when content block > x_number, it will display in the collapse).

    The Important area is the Content area (bottom). It will listing all collections content. Click to block to edit.

    You will got options: Upload image, Edit caption and Responsive layout. Select options to show as your idea.



    Collection Tab

    • Area: Right content

    • Section Title as add new: Collection Tab in Collection group

    • Section Title as existed in demo: What's New

    Click to edit.

    These option fields for this section: Display on, Title, Collection sub-title, Image size, Responsive columns group options, Products per row, Limit product show.

    The Important area is the Content area (bottom). It will listing all tabs content. Click to block to edit. If you want add more tab, click to

    In each tab content, you need input the Tab title and select Collection to show products.



    Banner Grid

    • Area: Right content

    • Section Title as add new: Banner Grid in Image group

    • Section Title as existed in demo: Banner Grid. There are 2 Banner Grid sections in Home page.

    Click to edit.

    These option fields for this section: Display on, Title (Heading), Sub-title, Image width and Image height.

    The Important area is the Content area (bottom). It will listing all banner(s) content. Click to banner block to edit. If you want add more banner block, click to

    In each block, you need upload Image banner, Link, Action text and Responsive group options.



    Collection Slider Column

    • Area: Right content

    • Section Title as add new: Collection Slider Column in Collection group

    • Section Title as existed in demo: Collection Slider Column

    Click to edit.

    These option fields for this section: Display on, Limit products to show, Product images size, Slider settings.

    The Important area is the Content area (bottom). It will listing all column(s) content. Click to column block to edit. If you want add more column block, click to

    In each block, you need input Title column and select Collection to show products.



    Collection Slider

    • Area: Right content

    • Section Title as add new: Collection Slider in Collection group

    • Section Title as existed in demo: RECENTLY VIEWED

    Click to edit.

    These option fields for this section: Display on, Collection title (Heading), Collection sub-title, Select Collection to show products, Slider and Responsive options.



    Left Sidebar

    • Area: Left sidebar

    • Section Title as add new: Fixed section. Can not add new/more.

    • Section Title as existed in demo: Left Sidebar

    Click to edit.

    Only 1 Section but include all blocks in Left sidebar. There are 6 block content:


    All Collections

    Input the Title block and select the Navigation to show link list.


    Banner

    Upload image as banner and set the link option.


    Testimonial

    There are options to control the slider options and Reviewer element options (Avatar, Name, Job, Review text). Currently the limit is 5 Reviewer element. If you want more, you can contact with Technical support.


    Article (From our Blog)

    There are options: Title block, select Blog to show Article, limit article and slider options.


    Products (Latest Products)

    There are options: Title block, select Collection to show products and limit appear.


    Instagram

    Need input the Title block and select method to show Instagram photos.

    There are 2 methods:

    • by API Access Token: This method is auto show photos from your account. You need input your account information in the group "Instagram API Access Token".

      • API Access Token: get one https://instagram.pixelunion.net/

      • User ID: this is User ID, not Username. The first string of Access Token array.

      • Number of images: limit photos.

    • by URL: this is manually method. You need input the Instagram photo URL into the group "Instagram Photo Url". The URL don't have the / at the end.

      • https://www.instagram.com/p/BoL5WAEhrTT ---> Right, worked.

      • https://www.instagram.com/p/BoL5WAEhrTT/ ----> Wrong, don't work.


    Bottom Banners

    • Area: Bottom

    • Section Title as add new: Fixed section. Can not add new/more.

    • Section Title as existed in demo: Bottom Banners

    Click to edit.

    It the same Banner Grid section.



    Bottom Information

    • Area: Bottom

    • Section Title as add new: Fixed section. Can not add new/more.

    • Section Title as existed in demo: Bottom Information.

    Click to edit.

    The Important area is the Content area (bottom). It will listing all Information block content. Click to block to edit: add icon image, input text content and select Responsive column.

    If you want add more block, click to



    Bottom Brand

    • Area: Bottom

    • Section Title as add new: Fixed section. Can not add new/more.

    • Section Title as existed in demo: Bottom Brand.

    Click to edit.

    There are options for Slider settings and Responsive columns.

    The Important area is the Content area (bottom). It will listing all Brand block content. Click to block to edit: add Brand image and link to.

    If you want add more block, click to
